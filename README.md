## Getting Started

### Installation


1. Working with legacy dependencies


`npm install --legacy-peer-deps && npx next dev`

2. First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Running on stackblitz:

- `npm i && npx next dev`
